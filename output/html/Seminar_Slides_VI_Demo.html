Framework tái sử dụng

Phần nào dùng lại được?

Đọc cấu hìnhTa lấy driver, địa chỉ CSDL và tên các file cần dùng.
Mở kết nốiJDBC kết nối Java với CSDL của project.
Nạp dữ liệuFile seed tạo sẵn dữ liệu ban đầu khi ta cần.
Kiểm traFile expected ghi lại dữ liệu ta mong đợi sau khi test.

Giữ nguyên

DatabaseTestConfig đọc cấu hình; JdbcDatabaseTestSupport làm việc với CSDL; ConfiguredDatabaseStateTest chạy test.

Thay theo project

Ta đổi JDBC driver, địa chỉ CSDL, file seed, file expected và các cột tự sinh cần bỏ qua.

Ta không cần viết lại khung test; ta chỉ chuẩn bị dữ liệu và cấu hình phù hợp với project mới.
